## System Requirements
* Ubuntu, Debian, CentOS or RHEL server running VestaCP 0.9.8-17+ with root access.
* Web Server with PHP 5.4+ installed (does not have to be VestaCP powered) and the following PHP extensions & options enabled:
 - PHP cURL
 - PHP OpenSSL
 - PHP gettext
 - PHP MySQLi
 - PHP FTP
 - PHP File Uploads
